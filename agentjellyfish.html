"Agent Jellyfish"
Interactive Narrative
Single Player Room Escape Game
"Agent Jellyfish" features the first mission, which is also a little entry test, of a young, newbie agent at The Lost Cape, a secret underwater organization. Controlling the pink jellyfish avatar, player has to explore the space and interact with different objects. By collecting information and clues, player needs to work out puzzles to get pass the entry test for Agent Jellyfish.

This game prototype is developed with p5.js and p5.play library.
I'm going to further develop this project later.
